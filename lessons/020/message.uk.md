🇪🇸 *Урок 20 — Language Transfer*

*Головна ідея:*
Тобі не потрібно вчити майбутній час, щоб говорити про майбутнє\! Просто використовуй теперішній час і додай слово\-підказку \(_mañana_, _pronto_\), або використовуй конструкцію _voy a_ \+ дієслово \(я збираюся\)\.

*Нові конструкції:*
Дієслово _ver_ \(бачити\) має неправильну форму "я" — _veo_\. Також додаємо корисні слова для майбутнього:

• _veo_ \(I see\) — я бачу
• _vemos_ \(we see\) — ми бачимо
• _pronto_ \(soon\) — скоро
• _nos_ \(us / each other\) — нас / один одного

• _nos vemos_ \(we see each other\) — побачимося
• _la semana_ \(the week\) — тиждень
• _que viene_ \(that comes\) — що приходить
• _la semana que viene_ \(next week\) — наступного тижня

*Запам'ятай:*
Фраза _nos vemos_ \(дослівно "ми бачимо один одного"\) — це найпопулярніший спосіб попрощатися іспанською\. Майбутній час тут мається на увазі автоматично\!

*Приклади:*
• _Nos vemos la semana que viene\._ — Побачимося наступного тижня\.
• _¿Por qué no vas a vernos pronto\?_ — Чому ти не збираєшся побачити нас скоро\?
• _La veo esta noche\._ — Я побачу її сьогодні ввечері\.
• _Hablamos mañana\._ — Поговоримо завтра\.

Далі: Як говорити про минуле \(не вчивши минулого часу\)
