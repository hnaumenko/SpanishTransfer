🇪🇸 *Lesson 74*

Ever notice how some Spanish verbs just refuse to play by the rules? Today you're learning one that literally changes its vowels when you stress them\.

Here's *elegir* \(to choose\)\. When you stress the "e," it becomes "i":
_Yo elijo_ — I choose
_Tú eliges_ — You choose
_Él elige_ — He chooses

But when the stress moves away: _nosotros elegimos_ — we choose\. No change\.

Now picture this:
— Why did you cook so much?
— *He cocinado para que ellos coman\.* I cooked so that they eat\.
— Smart\. *He hecho todo para que tú vuelvas\.* I've done everything so that you come back\.

Here are 8 phrases using *para que* \+ subjunctive:

_No quiero que elijas por mí_ — I don't want you to choose for me
_Hemos cocinado para que coman_ — We cooked so that they eat
_Lo he hecho para que vuelvas_ — I've done it so that you come back
_Ellos lo han hecho para que volvamos_ — They did it so we come back
_Trabajo para que vivas mejor_ — I work so that you live better
_Llamo para que sepas la verdad_ — I'm calling so you know the truth
_Escribo para que entiendas_ — I write so that you understand
_Salgo temprano para que llegues a tiempo_ — I leave early so you arrive on time

💡 After *para que*, you *must* use subjunctive\. It always implies "in order that/so that\."
